📌 なぜ await を使っているの？
solana.connect()（Phantom との接続）は、ユーザーの操作（許可）を待つ必要がある処理です。

Phantom が「この dApp に接続していいですか？」とユーザーに確認する → ユーザーが「OK」するまで時間がかかる

なので、await を使って「その処理が終わるまで、次の処理（アドレスのセット）を待ってね！」という意味になります。
